# Android weather app

## Deploy

- `app\build.gradle.kts`
```kts
versionCode = 3
versionName = "0.2.0"
```
2. Добавить коммит с номером версии в мастер `git commit -am "v0.2.0"`
3. Добавить тег с номером версии `git tag -a 0.2.0 -m "0.2.0"`
4. Запушить мастер с опцией `git push --follow-tags`
